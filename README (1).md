# 🧠❤️ Neuro-Cardiac Digital Twin

AI-powered Neuro-Cardiac Digital Twin integrating real-time ECG and EEG analytics for intelligent brain–heart monitoring, predictive cardiovascular risk assessment, and automated medical report analysis.

## Features
- Real-time ECG analysis
- EEG-based neural stress metrics
- HRV calculation (RMSSD, Heart Rate)
- Risk prediction model
- Dark futuristic AI UI
- PDF report generation

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NumPy
- WFDB
