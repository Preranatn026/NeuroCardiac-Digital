# Real-Time NeuroCardiac Digital Twin

## Introduction

Heart diseases and neurological disorders are major causes of death worldwide. In current healthcare systems, ECG (heart) and EEG (brain) are monitored separately, which makes it difficult to understand how both organs influence each other in real time. Many serious conditions such as stroke, stress-induced arrhythmia, and epilepsy-related cardiac issues are often detected too late.

This project aims to bridge that gap.


## Project Overview

Real-Time NeuroCardiac Digital Twin is an AI-based system that analyzes ECG and EEG signals together instead of separately. The goal is to understand heart-brain interaction and detect early warning signs before critical events occur.

The system collects ECG and EEG data, preprocesses the signals to remove noise, and extracts important features such as heart rate variability and brain wave patterns. A multimodal AI model then studies how these signals interact under normal conditions and detects abnormal patterns in real time.


## Key Features

- ECG and EEG signal preprocessing  
- Feature extraction from heart and brain signals  
- Multimodal AI-based analysis  
- Real-time risk detection and alerts  
- Simple dashboard for visualization  


## Technologies Used

- Python  
- NumPy  
- Pandas  
- Signal Processing  
- Machine Learning  


## Conclusion

By integrating cardiac and neural monitoring into a single system, this project moves healthcare from reactive treatment toward proactive and preventive care.
